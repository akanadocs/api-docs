OAuth GrantDetail Object

Contains information details about an OAuth grant.

This object is used by the following operations:

  • GET /oauth/auz/grants/{GrantID}/details

The properties included in the GrantDetail object are listed below.

Property Type Description
Grant Grant Contains information about an OAuth grant.
ResourceOwnerAuthenticationInstructions ResourceOwnerAuthenticationInstructions Contains information about the instructions for authenticating an OAuth resource owner.
