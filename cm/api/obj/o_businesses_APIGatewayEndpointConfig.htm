APIGatewayEndpointConfig Object

Contains configuration information for an API Gateway endpoint.

The properties included in the APIGatewayEndpointConfig object are listed below.

Property Type Description
APIEndpointDomain String The domain for the endpoint.
IncludeRandomHostname Boolean If set to true, a random hostname is created for the endpoint.
